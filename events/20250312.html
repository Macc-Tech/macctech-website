Creating More Inclusive Workplaces with Digital Tools - Lee Valentine

This talk, hosted by Nexer's Lee Valentine will explore practical tips to create more accessible and inclusive workplaces, with a focus on how digital tools can support this. We will see how .NET developers can create Mixed Reality experiences for headsets such as Meta Quest 2 & 3 using just C# & .NET with Visual Studio. This is a great way for .NET developers to get into Mixed Reality development.
